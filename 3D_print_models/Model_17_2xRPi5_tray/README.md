# Double fan with Double Raspberry Pi 5 Tray for DeskPi Rackmate T0/T1/T2

## Description 
This Tray is almost 2U rackmount for 2 Raspberry Pi 5 with 2 Fans for heat
dissipationing.

* Model Length: 10-inch
